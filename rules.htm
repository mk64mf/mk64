Rules

 

Gameplay

Cheats codes are not allowed in any way, every Setting in the Hack must be default except for Input Display which you can use if you want to or not. N64 and Emu is allowed but on Emu it is strongly prefered to submit with a Live recording and not the Replay because guess what! Save states are not allowed :D

Rom-Hack Time Trials only: Playing the NTSC rom on an PAL console is allowed

 

Proof

Video proof is only needed for times that are ranked 1st, For N64 it doesn't matter if it is a Live or Replay recording, on Emu it's strongly preferred to post a Live recording.
